🏡 One-time Setup

Connect your Supabase database so all family members share the same bookings. This takes about 5 minutes and only needs to be done once.

1 · Create a free Supabase project Go to supabase.com → New Project. Copy your Project URL and anon public key from Settings → API.
2 · Run this SQL in Supabase SQL Editor Go to SQL Editor in your Supabase dashboard and paste + run the SQL from the SETUP.sql file included with this app.
3 · Paste your credentials below and save These are stored only in your browser and never sent anywhere except Supabase.
